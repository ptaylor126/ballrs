## Display Rules

### Achievements Screen

- All achievements shown in a grid
- Unlocked: Full color + unlock date
- Locked (regular): Grayed out with description visible
- Locked (secret): Grayed out with "???" description
- Progress bar for countable achievements (e.g., "5/10 duels won")

### Level Up Celebration

When user levels up:
- Full-screen celebration modal
- Shows new level number and title
- Shows any new unlocks (frames, icons)
- Confetti animation

### XP Progress Bar

- Shown on home screen below username
- Shows current level + progress to next level
- Format: "Level 7 â€¢ 1,250 / 2,100 XP"
- Animates when XP is gained
